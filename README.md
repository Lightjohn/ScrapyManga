# ScrapyManga

## Usage

`scrapy crawl SpiderName -a start_url=URL1,URL2`

or 

`python3 start.py crawl SpiderName` (Not recommended)

## Available spiders
* mangapark
* mangarock
* mangazuki.online (soon)

## Notes

* Output path is set in `settings.py` see **FILES_STORE**
* Default log level is `INFO`
